# PostgreSQL

PostgreSQL은 오픈소스 관계형 데이터베이스(RDBMS)이다.

안정성과 확장성을 바탕으로  
대규모 서비스에서도 많이 사용되는 데이터베이스 중 하나이다.

---

## 1. PostgreSQL이란

PostgreSQL은 데이터를 테이블 형태로 저장하고  
SQL을 통해 데이터를 조회, 수정할 수 있는 관계형 데이터베이스이다.

트랜잭션을 지원하며, 데이터의 일관성과 무결성을 보장한다.

---

## 2. 특징

PostgreSQL은 다음과 같은 특징을 가진다.

- 오픈소스 데이터베이스 (무료 사용 가능)
- 표준 SQL을 잘 준수
- 다양한 확장 기능 지원 (Extension)
- 강력한 트랜잭션 처리 (ACID 보장)
- 복잡한 쿼리 처리에 강점

---

## 3. 장점

- 안정성이 높고 데이터 무결성 보장이 강함
- 복잡한 쿼리 및 데이터 처리에 유리
- 확장 기능을 통해 다양한 기능 추가 가능
- 대규모 서비스에서도 사용 가능

---

## 4. 단점

- MySQL에 비해 설정과 사용이 다소 복잡함
- 단순 조회 성능에서는 상대적으로 불리할 수 있음
- 초기 학습 난이도가 있는 편

---

## 5. 어디에 사용되는가

PostgreSQL은 다음과 같은 환경에서 주로 사용된다.

- 웹 서비스 백엔드 데이터베이스
- 데이터 분석 시스템
- 금융 및 안정성이 중요한 시스템
- 대용량 데이터를 다루는 서비스

---

## 정리

- PostgreSQL은 오픈소스 관계형 데이터베이스이다
- 안정성과 확장성이 뛰어나 다양한 서비스에서 사용된다
- 복잡한 쿼리와 데이터 처리에 강점이 있다
